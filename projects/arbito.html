Arbito

A first person horror game

October 2022
itch.io page

For a revival game jam, I wanted to revisit an earlier version of this game developed during another jam. I was getting familiar with Blender around this time, so I utilized it to create my own environment, rather than relying on Unity assets. Because I was tight on time, instead of rigging a 3D model I instead made the main enemy of the game a moving photo of Saul Goodman from this hit show Breaking Bad and Better Call Saul, complete with voice lines.
