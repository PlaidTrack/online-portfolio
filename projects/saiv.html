Safety Accessory for Industrial Vehicles

A protective system using Lidar technology

April 2023

The safety accessory for industrial vehicles (SAIV for short) is an attachment for heavy machinery, specifically for forklifts, that will alert the operator whenever a collision is imminent, with the goal of curbing workplace accidents. My main contributions to this project related to reviewing code and assisting with the production of the presentation video.


This was created as a submisstion to the TiM 10k Challenge, an annual intercollegiate contest held by Sick, a German-based global manufacturer of sensors and sensor products.
