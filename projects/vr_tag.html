VR Tag

A multiplayer game of tag

February 2024

A game of tag involves many people - so naturlaly including a multiplayer element would make such a game more complete. This was done using Photon for Unity, and this allows multiple people using VR headsets to play the game together. Each player's movement is shown in the other players' headsets in real time, which allows people to tag and interact with each other just like in a real game of tag. Not only does this project serve as an introduction to multiplayer functionality, but also an introduction to VR functionality using Unity's XR Interaction Toolkit.


This game takes place on a circular world, providing infinite space for players to run around as they game. This was done by creating gravity fields around spherical objects, creating a navigation technique similar to Super Mario Galaxy.
